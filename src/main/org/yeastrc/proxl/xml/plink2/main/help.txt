Description: Convert the results of a pLink 2.x analysis to a ProXL XML file suitable for import into ProXL.
             Designed to work only with single, label-free searches that used a single cross-linker.

Usage: java -jar plink2toProxlXML.jar -p path -o path -f path [-b path ] [-r path]

Example: java -jar plink2toProxlXML.jar -p c:\plink_run\run_name.plink -o "c:\out put\mySearch.proxl.xml" -f c:\fastas\myFasta.fasta
         java -jar plink2toProxlXML.jar -p c:/plink_runs/search/run_name.plink -o c:/output/mySearch.proxl.xml c:/fastas/myFasta.fasta

         Long example:
         java -jar plink2toProxlXML.jar -p C:\Users\PLink User\Desktop\gTuSC\pLink_test\pLink_test.plink ^
                                        -o C:\Users\User\Desktop\gTuSC.proxl.xml ^
                                        -f C:\fastas\myFasta.fasta ^
                                        -b C:\pFindStudio\pLink\2.3.0\bin ^
                                        -r C:\Users\User\Desktop\gTuSC\pLink_test\reports

Options:
	-p	[Required] Full path to pLink 2 parameters file used in the search (ends in .plink).
	-o	[Required] Full path to use for the ProXL XML output file (including file name).
	-f	[Required] Full path to FASTA file used in the experiment.
	-b	[Optional] Full path to the pLink installation directory, where modification.ini and xlink.ini may be found.
		           This defaults to: C:\pFindStudio\pLink\$VERSION\bin
		           Where $VERSION is found in the parameters file.
	-r	[Optional] Full path to the data reports directory for pLink 2 results. This directory 
		           contains run_name.date.csv (e.g. my_plink_search_2018.02.25.csv)
		           If not present, value from the parameters file will be used.
